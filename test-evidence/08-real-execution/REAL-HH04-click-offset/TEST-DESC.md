# T-HH-04: 拟人点击（偏移）

## 测试描述
验证 HumanHand 的点击偏移机制：点击在目标 bbox 中心附近有随机偏移，多次点击不完全相同。

## 测试步骤
1. 定义目标 bbox: (500, 400, 600, 450)
2. 3 次点击 bbox 中心附近（加入 [-5, +5]px 随机偏移）
3. 记录每次点击坐标
4. 验证每次点击坐标都在 bbox 内，且三次不完全相同
5. 截屏保存

## 测试参数
- 目标 bbox: (500, 400, 600, 450)
- bbox 中心: (550, 425)
- click_offset_range: 5px

## 点击记录
| 次数 | 坐标 | X偏移 | Y偏移 | 在bbox内 |
|------|------|-------|-------|---------|
| 1 | (547, 425) | -3 | 0 | YES |
| 2 | (549, 425) | -1 | 0 | YES |
| 3 | (548, 427) | -2 | 2 | YES |

## 验证结果
- 所有点击在 bbox 内: YES
- 有随机偏移（三次不完全相同）: YES
- **总判定: PASS**
