# T-HH-03: 近距离微调移动

## 测试描述
验证 HumanHand 在近距离（~50px）移动时，有细微抖动效果，终点精度在 +/-5px 内。

## 测试步骤
1. 使用 WindowsAdapter 大距离移动到起始位置 (500, 500)
2. 截屏记录起始状态
3. 使用 HumanHand 的贝塞尔曲线 + jitter 逻辑进行近距离微调移动到 (550, 510)
4. 最终精确移动到目标点
5. 记录鼠标实际终点位置
6. 验证终点偏差 <= 5px 且轨迹有抖动
7. 截屏记录结束状态

## 测试参数
- 起始位置: (500, 500)
- 目标位置: (550, 510)
- 移动距离: 51.0px
- jitter_range: 2.0px
- move_speed: 1.0
- bezier_control_points: 3
