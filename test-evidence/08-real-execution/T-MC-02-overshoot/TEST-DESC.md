# T-MC-02: 过冲回调

## 测试目的
验证 MotorController 闭环控制在过冲场景下能回调修正。

## 测试场景
1. 过冲回调修正成功：第1次点击后目标仍在（过冲），第2次修正成功
2. 持续过冲达到最大重试：目标始终存在，最终返回 VERIFICATION_FAILED
3. 无验证模式：跳过验证，一次成功

## 测试结果
- 场景1 过冲修正: PASS (attempts=2)
- 场景1 点击次数验证: PASS
- 场景2 持续过冲: PASS (attempts=3)
- 场景3 无验证模式: PASS
- 通过: 4/4
- 状态: PASS
