# T-PF-07: 鼠标拖拽测试

## 测试目的
验证 WindowsAdapter.drag_to() 方法能正确执行鼠标拖拽操作。

## 测试参数
- 起点: (500, 500)
- 终点: (800, 500)
- 持续时间: 0.5s
- 按钮: left (默认)
- 屏幕: 3840x2160

## 预期结果
1. 鼠标从起点拖拽到终点
2. 拖拽完成后鼠标位置偏差 <= 10px

## 实际结果
- 拖拽后鼠标位置: Point(x=800, y=500)
- 偏差: dx=0, dy=0
- 截图: screenshot_after_drag.png (已保存)
- **[PASS]** 测试通过
