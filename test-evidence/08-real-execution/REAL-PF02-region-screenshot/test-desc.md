# T-PF-02: 截屏指定区域

## 测试目的
验证 WindowsAdapter.screenshot(region=...) 能正确截取屏幕指定区域。

## 测试参数
- region: (560, 480, 800, 600) — 屏幕中心 800x600 区域

## 预期结果
- 返回有效 PNG 字节
- 图片尺寸 = (800, 600)
- 文件大小 > 10KB

|--------|------|------|------|
| PNG 签名 | `\x89PNG` 开头 | `\x89PNG` 开头 | PASS |
| 文件大小 | > 10KB | 719,820 bytes (~703KB) | PASS |
| 图片尺寸 | (800, 600) | (800, 600) | PASS |

**结论：T-PF-02 测试通过。** WindowsAdapter.screenshot(region=...) 能正确截取指定区域，返回有效 PNG 字节，尺寸与 region 参数一致。
