## 根因分析

### 失败链条
```
搜索框定位失败(LOCATION) → 关键词未输入 → 回车在错误位置(OPERATION) → 无搜索结果 → 坐标点击命中游戏窗口(LOCATION)
```

### 关键问题
1. **固定坐标不可靠**：必应搜索框在 3840×2160 分辨率下位于屏幕中上部，但 Cookie 隐私弹窗遮挡了搜索框区域
2. **无视觉验证**：脚本判定"通过"但实际操作全部偏离，因为 pyautogui 不检查操作结果
3. **遮挡/弹窗干扰**：真实环境中经常有弹窗（Cookie、广告、通知），固定坐标无法应对
4. **窗口焦点不确定**：多个窗口叠放时，pyautogui 点击可能命中非目标窗口

## 关键洞察

> **没有 UI 元素精确检测能力（UIA/OCR），固定坐标方案在真实桌面环境中完全不可靠。**
